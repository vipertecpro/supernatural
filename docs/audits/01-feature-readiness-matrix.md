## Prompt 2 Readiness Gate

Prompt 2 should stop when the following are true: the intended starter/package/configuration tree is reproducible; all existing checks pass; email verification is actually enforced; neutral roles/policies/admin/moderator/audit boundaries exist; a minimal universe/source/license/spoiler foundation is tested; a versioned API convention is documented and tested; Reverb/environment defaults are safe; and minimum open-source/content-rights documentation exists. No community, chat, immersive design, or Supernatural content ingestion is required to satisfy that gate.
