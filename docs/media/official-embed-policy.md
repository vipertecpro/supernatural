# Official external embed policy

The cinematic public experience accepts only published, public, moderated YouTube `ExternalEmbed` records from an official or authorized channel. `provider_metadata.authorized_channel` must be true and the latest source rights review must permit embedding.

URLs are normalized by the existing provider allowlist. Public presentation uses `www.youtube-nocookie.com`, creates no iframe until the visitor clicks, retains official controls, never autoplays hidden audio, never downloads or clips video, and links to the canonical provider page. When no approved record exists, an original signal visual replaces the player.
